Oiyo 是以融合AI协作，提升开发体验，降低心智成本为三大理念的 UniApp 增强型工程框架

通过使用约定和主观的工程结构来解决以往重复的任务流程，让开发者得以专注业务的功能开发，其拥有：

- 根上下文
- 布局系统
- 自动路由
- 页面元信息声明
- 跳转守卫
- 主观工程结构
- 稳定底层依赖版本
- 一站式团队服务
